Smart Learning

This optional mode learns how your apps behave and adapts future nap passes without sending data anywhere.

  • Profiles stay local on this PC and are based on process name/path, memory use, CPU bursts, and wake events.
  • Apps que você usa de novo com frequência recebem um perfil mais leve, então chats, navegadores e ferramentas voltam mais rápido.
  • When memory pressure rises, heavy idle background apps can receive a stronger nap automatically.
  • Não fecha apps, não altera drivers, não força planos de energia e não envia telemetria.

Administrator pass

Some apps refused process changes because Windows requires higher permission. Smart Nap can ask UAC and run one elevated optimization pass.

  • This does not keep Smart Nap permanently elevated. It only runs one administrator pass.
  • Não instala drivers, serviços nem força planos de energia.
  • Some protected apps can still refuse changes even with administrator permission.
