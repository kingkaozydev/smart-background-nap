# Smart Background Nap 0.4.7

Update-control refinement release: fixes the user-managed automatic update toggle, removes noisy PC Profile copy, and includes the deployment manifest for the public site.

## Highlights

- Single-file release: download and run SmartBackgroundNap.exe.
- Fixed the automatic update preference so users can turn automatic release reminders on or off from the update popup.
- The manual "Check updates" action still works even when automatic checks are disabled.
- Removed the PC Profile snapshot label from the launcher.
- Included the package manifest required to deploy the public Smart Background Nap site.

## Trust Notes

Smart Background Nap has no telemetry, no user data uploads, no driver install, no service install, no startup registry key, and no permanent administrator elevation. The launcher can check the official GitHub Releases endpoint only when automatic checks are enabled or when the user manually checks for updates.
