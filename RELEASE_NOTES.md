# Smart Background Nap 0.4.13

Fixes manual app policy selection in the Live Manager.

## Changes

- A/P/L/B/D buttons now update immediately after clicking.
- Manual policies are matched by executable path and process name.
- The dashboard keeps the selected policy visible before the next engine pass.
- Multi-process apps such as browsers and Electron apps should behave more consistently.
